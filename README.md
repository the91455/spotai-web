# SpotAI - AI Songwriting Companion

SpotAI is a modern, web-based tool designed to act as your co-pilot for creating songs with generative audio platforms like Suno AI. 

Built with **Vanilla HTML/CSS/JS** and powered by Google's **Gemini Flash 2.0 API**, it wraps complex prompt engineering into a premium, Spotify-inspired interface.

## Setup

1. Clone the repository.
2. Open `index.html` in your browser (or serve it with a simple server like `python3 -m http.server`).
3. Click the **API** button in the header.
4. Enter your [Google Gemini API Key](https://aistudio.google.com/app/apikey).
5. Start creating!

## Tech Stack

- **Frontend:** Vanilla JS, CSS (Grid/Flexbox), HTML5.
- **AI:** Google Gemini API (Client-side integration).
- **Icons:** Custom SVG paths.
